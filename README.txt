# PIPE - Ambiente di Sviluppo per la Creazione di Reti di Petri

![Esempio Rete di Petri](https://github.com/Langiott/PIPE/blob/28836f9481345a3b04642acaeb4fe90eeae91b6b/IMG22.png)

## Descrizione
PIPE è un ambiente di sviluppo che ti permette di creare e simulare reti di Petri. Le reti di Petri sono un potente strumento per modellare e analizzare sistemi concorrenti, come impianti manufatturieri, macchine per il caffè o sistemi di parcheggio per automobili.

## Istruzioni per l'Utilizzo
Per far partire il programma, segui questi passaggi:

1. Scarica tutti i file necessari, compresi gli esempi se desideri.
2. Metti tutti i file in una cartella dedicata.
3. Esegui il file .exe.
4. Il programma si aprirà dalla prompt dei comandi, dove potrai configurare le impostazioni desiderate.
5. Successivamente, si aprirà l'applicazione Pipe.

È importante notare che non è possibile salvare file direttamente all'interno dell'applicazione Pipe.
